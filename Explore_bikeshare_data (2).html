Question 1¶

What are the counts of each user type?

In [6]:
## Function to get counts of each user type
## Convert to table to get counts and convert back to dataframe
## remove blank space user types
usertype_counts = function(dataset, column) {
  column_counts = table(dataset[[column]])
  column_counts = as.data.frame(column_counts)
  colnames(column_counts) = c(column, "Counts")
  column_counts = subset(column_counts, column_counts[[column]] != "")
  return(column_counts)
}
